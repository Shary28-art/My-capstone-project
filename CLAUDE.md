# AI Assistant Guidelines for My Capstone Project

## Project Overview
This repository contains the foundation for a frontend-driven AI engineering capstone project.

## Tech Stack & Environment
- **Runtime:** Node.js (LTS)
- **Package Manager:** npm (or change to pnpm/yarn if you use them)
- **Version Control:** Git

## Build & Development Commands
- Install dependencies: `npm install`
- Run development server: `npm run dev` (update this once your framework is set up)
- Build project: `npm run build`

## Code Style & Preferences
- **Architecture:** Modular, component-driven frontend architecture.
- **Formatting:** Use clean, readable code with descriptive variable names.
- **Git Strategy:** Always use **Conventional Commits** format for all code changes.
  - Examples: `feat: add chatbot component`, `fix: resolve api timeout`, `docs: update setup guide`.
